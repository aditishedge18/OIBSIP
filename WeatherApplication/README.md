# Weather Application

A simple Python weather application using OpenWeatherMap API.

## Features
- Shows city weather
- Shows temperature
- Shows humidity
- Shows weather condition
- Shows country

## Technologies Used
- Python
- Requests Library
- OpenWeatherMap API

## How To Run
1. Install requests library
2. Run weather.py
3. Enter city name

## Example
City: Mumbai
Temperature: 32°C
Weather: Haze
